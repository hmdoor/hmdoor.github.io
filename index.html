Fun Facts!

I received my first personal computer when I was in 3rd grade and it was then that my love of technology began. I got my first taste of HTML back when MySpace was a big deal and really enjoyed being able to personalize my own page. I didn't get back into it until recently when I discovered I could combine my love of design and technology into a vibrant career. This curiosity led me to TechElevator where I'm learning Java ,HTML, CSS, JavaScript, JSP, Spring Web MVC, JDBC, PostgreSQL, unit testing (JUnit) and Git.

Before I discovered my passion for web development I graduated from Otterbein University with a BA in Music and Business then completed a year long fellowship volunteering and working in Cleveland. During that time I worked with the Cleveland Leadership Center where I learned how to work with people across all sectors, ages and from varying socioeconomic and leadership backgrounds. It was there where I started picking up small design projects and become interested in learning to use the Adobe creative suite. I've also picked up some small photography projects on the side which has been a fun way to learn Photoshop and Lightroom.

My Projects

National Park Weather Service:

For this capstone project, my partner and I created a multi-page website (using Spring Web MVC) where users are able to select from a list of national parks to view the park details and weather in celsius or fahrenheit. Users are also able to add surveys for the parks. Once the survey is submitted, it is stored in a database (Postgres) and the user is redirected to a page that displays all survey results, pulled in from the database.

National Park Registration Application:

In this capstone project, my partner and I created a command line application for users to search for national parks, view their campgrounds, check reservation availability and make a reservation. All the data was stored in a database (Postgres) and we used JDBC to access the data.

Vending Machine Application:

For this capstone, my partner and I created a command line vending machine application where users were able to view a list of items, using File I/O, see how many items were left, select an item and pay for it.

Java-Jam: Princess Edition

This is a side project I've modified to really understand the principals of object oriented programming. In this command line game, users are able to choose from a list of 10 different Disney princess, assign them a weapon and watch as they fight to the death! This project was created in Eclipse to run as a Java application.
